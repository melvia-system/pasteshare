- Get a anonymous link to share
- No registration required
- No tracking
- No ads

## Tech Stack
- Frontend: [Nuxt 3](https://nuxt.com)
- Backend: [Go](https://golang.org)
- Database: [PostgreSQL](https://www.postgresql.org)
- Development Tools: [Docker](https://www.docker.com)
